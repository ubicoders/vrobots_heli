# vrobots_heli


## Install Ubicoders package
```
pip install ubicoders-vrobots
```

## Run the virtual robot bridge
```
python -c "import ubicoders_vrobots.vrobots_bridge.vrobots_bridge_gui as gui; gui.gui_app_run()"
```
or
```
python bridge.py
```

## Connect
the sim from the web

python my_helil.py
